Creates Possibilistic Membership directly by comparing two fuzzy numbers. The operation is fn_a > fn_b.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

possibilistic_strict_exceedance(fn_a, fn_b)

Arguments

fn_a Fuzzy number.
fn_b Fuzzy number.

Examples

  • possibilistic_strict_exceedance(fuzzy_number_triangular(1.9,2.8,3.5),fuzzy_number_triangular(1,2,3)) →
  • possibilistic_membership_as_text(possibilistic_strict_exceedance(fuzzy_number_triangular(1.9,2.8,3.5),fuzzy_number_triangular(1,2,3))) → 'PossibilisticMembership(possibility: 0.88235294117647, necessity: 0.421052631578947)'
