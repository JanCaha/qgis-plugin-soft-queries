Creates readable string Possibilistic Membership representation.

Syntax

possibilistic_membership_as_text(possibilistic_membership)

Arguments

possibilistic_membership Possibilistic membership.

Examples

  • possibilistic_membership_as_text(possibilistic_membership(0.8,0.6)) → 'PossibilisticMembership(possibility: 0.8, necessity: 0.6)'
