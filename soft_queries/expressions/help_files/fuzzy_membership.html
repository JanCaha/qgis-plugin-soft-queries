Creates Fuzzy Membership directly from membership value.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

fuzzy_membership(value)

Arguments

value Numeric value of membership, should be from range [0, 1].

Examples

  • fuzzy_membership(0.5) →
  • fuzzy_membership_as_text(fuzzy_membership(0.5)) → 'FuzzyMembership(0.5)'
