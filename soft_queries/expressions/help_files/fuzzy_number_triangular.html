Creates triangular fuzzy number.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

fuzzy_number_triangular(min, kernel, max)

Arguments

min Minimal value of fuzzy number.
kernel Kernel value of fuzzy number.
max Maximal value of fuzzy number.

Examples

  • fuzzy_number_triangular(1, 2, 3) →
  • fuzzy_number_triangular(-1, 0, 1) →
  • fuzzy_number_as_text(fuzzy_number_triangular(-1, 0, 1)) → 'Fuzzy number with support (-1.0,1.0), kernel (0.0, 0.0) and 0 more alpha-cuts.'
