Creates triangular fuzzy number.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

fuzzy_number_triangular(min, kernel_min, kernel_max, max)

Arguments

min Minimal value of fuzzy number.
kernel_min Minimal kernel value of fuzzy number.
kernel_max Maximal kernel value of fuzzy number.
max Maximal value of fuzzy number.

Examples

  • fuzzy_number_triangular(1, 2, 3, 4) →
  • fuzzy_number_triangular(-1, -0.5, 0.5) →
  • fuzzy_number_as_text(fuzzy_number_triangular(-1, -0.5, 0.5)) → 'Fuzzy number with support (-1.0,0.5), kernel (-0.5, -0.5) and 0 more alpha-cuts.'
