Creates Possibilistic Membership directly by comparing two fuzzy numbers. The operation is fn_a >= fn_b.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

possibilistic_exceedance(fn_a, fn_b)

Arguments

fn_a Fuzzy number.
fn_b Fuzzy number.

Examples

  • possibilistic_exceedance(fuzzy_number_triangular(1,2,3),fuzzy_number_triangular(1.5,2.4,3.5)) →
  • possibilistic_membership_as_text(possibilistic_exceedance(fuzzy_number_triangular(1,2,3),fuzzy_number_triangular(1.5,2.4,3.5))) → 'PossibilisticMembership(possibility: 0.789473684210526, necessity: 0.263157894736842)'
