Creates readable string from Fuzzy Number, Fuzzy Membership and Possibilistic Membership.

Syntax

sq_as_string(sq_object)

Arguments

sq_object Fuzzy Number, Fuzzy Membership, Possibilistic Membership.

Examples

  • sq_as_string(fuzzy_number_triangular(1,2,3)) → 'Fuzzy number with support (1.0,3.0), kernel (2.0, 2.0) and 0 more alpha-cuts.'
  • sq_as_string(fuzzy_membership(0.5)) → 'FuzzyMembership(0.5)'
  • sq_as_string(possibilistic_membership(0.8,0.6)) → 'PossibilisticMembership(possibility: 0.8, necessity: 0.6)'
