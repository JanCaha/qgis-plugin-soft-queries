Creates Possibilistic Membership directly by comparing two fuzzy numbers. The operation is fn_a =< fn_b.

The output cannot be properly represented as Expression outcome. It is only used in other expressions.

Syntax

possibilistic_undervaluation(fn_a, fn_b)

Arguments

fn_a Fuzzy number.
fn_b Fuzzy number.

Examples

  • possibilistic_undervaluation(fuzzy_number_triangular(1,2,3),fuzzy_number_triangular(1.5,2.2,2.5)) →
  • possibilistic_membership_as_text(possibilistic_undervaluation(fuzzy_number_triangular(1,2,3),fuzzy_number_triangular(1.5,2.2,2.5))) → 'PossibilisticMembership(possibility: 1.0, necessity: 0.384615384615385)'
